A premium interactive geography app built around a cinematic 3D Earth globe.
Explore countries, search instantly, compare nations, and save favorites in one polished experience.

## Features

* Interactive 3D globe exploration
* Country search and instant focus
* Country detail view with key statistics
* Side-by-side country comparison
* Personal bookmarks collection
* Dark / light theme support
* Smooth motion and polished UI transitions
* Persistent user preferences and saved state

## Installation

```bash
npm install
npm run dev
```
